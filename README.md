# eslint-plugin-vitest-sample

## 概要

- [eslint-plugin-vitest](https://www.npmjs.com/package/eslint-plugin-vitest)を使用したサンプルプロジェクトです。
- vitest用のeslintルールを実行します。

## 動作環境

- Node.js - 18.x
- Yarn - 1.22.x

## ライブラリ

- Vitest - 0.34.x
- ESLint - 8.47.x
- eslint-plugin-vitest - 0.2.x

## 動作確認

```bash
# パッケージインストール
yarn

# eslint実行
yarn lint
```
